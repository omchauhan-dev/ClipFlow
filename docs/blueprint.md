# **App Name**: ClipForge Studio

## Core Features:

- AI Content Generation Engine: Utilize advanced generative AI models to create videos, images, or audio based on user textual prompts and optional image inputs. This feature employs specific AI models as a tool for diverse content outputs.
- Dynamic Model and Mode Selection: Allow users to select from a range of specialized AI models (e.g., LTX-2.3 for HD Lip Sync) and distinct creation modes (e.g., Talking Actor, Scene Video, Character Image) via a dropdown and horizontal pill buttons, influencing generation parameters and outputs.
- Contextual Prompting Interface: Provide a fixed-bottom prompt bar with a textarea that adapts its placeholder text based on the selected generation mode. Includes an optional image upload mechanism for relevant modes.
- Interactive Output Display & Management: Render generated videos or images on a large central canvas. This includes loading states with progress bars, video playback controls, and options to download completed content.
- Real-time Generation Status & Feedback: Display crucial real-time information such as generation progress, estimated time remaining, current status messages, character count for prompts, and generation cost.
- Collapsible Generation History: Provide a right-side collapsible sidebar showcasing a list of past generations with thumbnails, mode icons, models used, and timestamps, allowing users to revisit or download previous outputs.

## Style Guidelines:

- The dominant color scheme is a deep, professional dark mode. Background is deep obsidian '#0a0e14', with layers of dark surfaces at '#0f141a' and '#1b2028' for visual hierarchy, conveying depth and sophistication. This foundation is suitable for a studio tool experience.
- The primary interaction color and brand accent is a vibrant Phosphor Green '#6bfe9c', used for active elements, highlights, and status indicators. This adds a tech-forward, energetic glow against the dark canvas. Text is a bright, clear '#f1f3fc', ensuring readability, with muted elements in a subtle '#8a8f9a'.
- Headings are set in 'Manrope' (sans-serif) with a bold weight (800) for strong visual impact and clarity, aligning with a modern, tech-oriented aesthetic. Body text uses 'Inter' (sans-serif) for its high readability and versatile application in digital interfaces. Note: currently only Google Fonts are supported.
- Icons are minimalist and consistent, predominantly outline or solid representations (e.g., ⬡, 🎭, ▶, 📎) that maintain visual coherence with the clean, modern interface. They are used contextually to represent modes, actions, and states, providing intuitive navigation and feedback.
- The application features a full-screen, single-page interface with a fixed top bar for branding and global controls, a dynamic central canvas for content display, a fixed bottom bar for generation controls, and a collapsible right-side panel for history. The layout prioritizes clear functional zones and immersive content presentation, employing 'no borders' for a sleek, unified appearance achieved through tonal depth.
- Subtle animations enhance user experience without distraction. These include a pulsing green status dot and progress bars during generation, a graceful fade-in effect for completed outputs, and a gentle green glow on hover for interactive elements, contributing to a premium, responsive feel.
